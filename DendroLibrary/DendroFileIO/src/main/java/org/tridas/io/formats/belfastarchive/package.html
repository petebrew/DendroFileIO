BelfastArchive is an extension of the BelfastApple format design by Martin Munro. It improves upon its predecessor by adding a number of metadata fields at the end of the file. This additional data replaces the last line (containing user and date) with something like the following: "[[ARCHIVE]]" 1277 9177 .01 1.035795 0.212144 IAN 21/01/96 TWYNING CHURCH #01 Pith F Sap 32 "" "[[ END OF TEXT ]]" The precise meaning of each line is as follows:

  • Start year (standard Gregorian calendar)
  • ?
  • Units
  • ?
  • ?
  • User 'space' date (dd/mm/yy)
  • Series name
