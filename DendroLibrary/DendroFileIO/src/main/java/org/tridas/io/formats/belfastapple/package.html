This file format was written by Jon Pilcher at Queen's University Belfast. The name is indicative of its use on Apple II computers.

It is an extremely simple 'data-only' format consisting of:

  • Line 1 - Series name
  • Line 2 - Sample number
  • Lines 3+ - Integer data values one per line typically (always?) in 1/100th mm units
  • Final line - User name followed by date in format dd/mm/yy
